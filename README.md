# AI CCTV Data Management System

ระบบจัดการข้อมูลสำหรับการติดตั้งกล้อง AI CCTV พัฒนาด้วย Python และ Streamlit

## ฟีเจอร์หลัก
1. **เพิ่มข้อมูลใหม่:** บันทึกชื่อบริษัท, ประเภทรถ, ตำแหน่งกล้อง และความยาวสาย
2. **ระบบฐานข้อมูล:** ใช้ SQLite ในการเก็บข้อมูล (ไฟล์ `cctv_data.db`) ข้อมูลไม่หายเมื่อปิดโปรแกรม
3. **ระบบค้นหา:** ค้นหาด้วยชื่อบริษัทเพื่อดูรายละเอียดความยาวสายและตำแหน่งติดตั้ง
4. **สรุปผล:** แสดงตารางข้อมูลที่สะอาดตา พร้อมปุ่มดาวน์โหลดไฟล์ CSV

## วิธีการติดตั้งและรันโปรแกรม

1. **ติดตั้ง Python:** ตรวจสอบว่าในเครื่องมี Python ติดตั้งอยู่ (แนะนำเวอร์ชัน 3.8 ขึ้นไป)
2. **ติดตั้ง Library ที่จำเป็น:**
   เปิด Terminal หรือ Command Prompt ในโฟลเดอร์ของโปรแกรม แล้วพิมพ์คำสั่ง:
   ```bash
   pip install -r requirements.txt
   ```
3. **รันโปรแกรม:**
   พิมพ์คำสั่ง:
   ```bash
   streamlit run app.py
   ```

โปรแกรมจะเปิดขึ้นมาใน Web Browser โดยอัตโนมัติ (ปกติจะเป็นที่อยู่ `http://localhost:8501`)
